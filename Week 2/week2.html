Ch. 2 Programming Basics

Template literals are special types of strings. Use the backtick ` to wrap the string instead of ". They also allow for interpolation in JS. Place the JS expression inside curly braces with the $ in front. ${ } The expression is evaluated and the result is returned in the string output.

Converting numbers to strings: preferred way is String(). parseInt() can be used to convert a string representation of a numerical value back to a number.

Ch. 3 Arrays, Logic, Loops

Create an array: const myArray = [ ]; or const myArray = new Array(); (not safe because you can override the value of Array).

Adding elements: heroes[0] = 'Superman';

Creating array literals: const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk']. Arrays can contain different data types.

Removing values: delete avengers[3]; This will remove the value but will be replaced with undefined. The array will still contain 4 elements.

Ch. 4 Functions

Ways to declare a function:
Function declaration: function hello() { }
Function expressions: const goodbye = function() {} - assigns anonymous function to variable

Invoking a Function:
hello();
goodbye() - parentheses are needed even if the function is assigned to a variable

Return values can be specified using the return keyword.

Parameters and Arguments: parameters are set when a function is defined, arguments are provided when the function is invoked.

Variable number of arguments: functions have special variable called arguments, an array like object that contains every argument passed to the function.
A better option than arguments is the rest operator. To use place three dots in front of last parameter (...args)

Default parameters: values used if no arguments are provided. function hello(name='World') {}. Put all default parameters after other parameters.

Arrow functions: => parameters come before the arrow and the main body of the function comes after. Multiple parameters go inside parentheses

Callbacks: a function that is passed to another funtion.

Array Iterators: methods that use callbacks to make them more flexible
forEach()
map()
reduce()
filter()
