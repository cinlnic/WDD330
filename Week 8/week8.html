Transforms and Transitions

Transforms

Tranform - lets you translate, rotate, scale, and/or skew elements (transforms don't work on inline elements)
to declare multiple transformations use a space-separated list of tranform functions
transform: translateX(40px) scale(1.5)
Order of the tranform functions matter

Translate - move elements up, down, left, right
transform: translate(x, y);

Scaling - scales an element by the factors horizontally then veritcally
transform: scale(1.5, 0.25);

Rotate - rotates an element around the point of origin by a specified angle value
transform: rotate(10deg);

Skew - specifies a skew along the x and y axes
transform: skew(15deg, 4deg);

Orgin of the transform can be changed
tranform-origin: 0 0;

Transitions

How to create a simple transition:
1. Declare the original state of the element in the default style declaration
2. Declare the final state of your transitioned element
3. Include the transition functions in your default style declaration

Transition functions
transition-property - defines CSS property that should be transitioned
transition-property: transform, color;
transition-duration - sets how long the transition will take
transition-timing-function - lets you control the pace of the transition
transition-timing-function: ease-out;
transition-delay - set a delay before the transition begins
transition shorthand - transition: transform .02s ease-out 50ms;
