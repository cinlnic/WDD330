Week 7 Notes

Further Functions

Functions Properties and Methods

Functions are first class objects - they can have properties and methods themselves

Call and Apply Methods
call() method can be used to set the value of this inside a function to an object that is provided as the first argument.
apply() works the same as call, expcept the arguments are provided as an array

Custom properties are added to functions the same way as to objects

Immediately Invoked Function Expressions

an anonymous function that is invoked as soon as it's defined
can be used to assign temporary variables, initialization code, using strict mode, creating self contained code blocks

Functions that Define and Rewrite Themselves

Functions can call, define, and rewrite themselves. This is done by assigning an anonymous function to a variable that the same name as the funtion

Recursive Functions

invokes itself until a certain condition is met

Callbacks

Can be used to facilitate event driven asynchronous programming

Promise
represents the future result of an ansynchronous operation
created using a constructor function

Closures

a reference to a variable that was created inside the scope of another function, but is then kept alive and used in another part of the program

Ajax

a technique that allows web pages to dynamically update without reloading

acronym for Asynchronous JavaScript XML

The Fetch API

standard for requesting and sending data asynchronously across a network

global fetch() method with a mandatory argument, the URL you want to fetch

Response Interface - deals with the object that's returned when the promise is fullfilled
has properties and methods that allow us to process the response effectively

Request Interface - more control over the request by providing a request object as an argument
created using the Request() constructor

Headers Interface - can be used to create a headers object, then added as a property of Request and Response objects
